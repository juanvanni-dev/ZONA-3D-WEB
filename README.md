# Zona 3D 🖨️

Landing page for **Zona 3D** — a 3D printing studio based in Santa Fe, Argentina, specializing in custom figures, spare parts, prototypes and personalized designs with 0.1mm precision.

**[→ Live Site](https://zona3d.vercel.app/)**

---

## About This Project

A real client project. Zona 3D needed a professional web presence that matched the quality and boldness of their work — a site that could attract new customers, showcase their gallery and drive direct contact via WhatsApp, Instagram and MercadoLibre.

The design language is strong and industrial: full uppercase, high contrast, minimal layout — reflecting the precision and boldness of 3D printing.

---

## Features

- 🖼️ **Product gallery** — categorized work showcase (lighting, industrial, gaming, collectibles)
- 📲 **WhatsApp contact integration** — direct link to start a conversation instantly
- 🛒 **MercadoLibre store link** — connects visitors to their online store
- 📱 **Fully responsive** — mobile-first layout
- ✨ **Scroll animations** — sections animate on entry
- ⚡ **Fast loading** — no frameworks, no unnecessary dependencies
